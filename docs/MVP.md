# ClubOps - MVP (Version 1)
Version: 0.1

Status: Planning

Last Updated: 7 July 2026

## Goal

Build a stable event management platform capable of managing an event from planning to post-event reporting without relying on advanced AI or automation.

---

## Core Features

### Authentication
- User Login
- User Logout

### Event Management
- Create Event
- Edit Event
- Delete Event
- View Event Details
- Event Timeline

### Budget & Finance
- Manual Budget Planning
- Expense Tracking
- Budget Buffer Calculation
- Budget Summary

### Resource Management
- Resource Inventory
- Check Resource Availability
- Procure Required Resources (Manual Entry)
- Resource Allocation

### Documentation
- Upload Event Photos
- Store Event Details
- Store Attendance
- Generate Report (Template-Based)
- AI-assisted Event Summary

### Attendance
- Manual Attendance Entry

### Organization Profile
- Hardcoded Organization Configuration
- Hardcoded Approval Workflow
- Hardcoded Report Template

---

## AI Features (Included)

- AI-generated Event Summary
- AI-generated Report Highlights

> AI only generates text.
> It never generates or modifies factual event data.

---

## Out of Scope (Version 1)

- AI Budget Planning
- AI Vendor Recommendation
- Adaptive Organization Profiles
- Learning AI
- Advanced Analytics
- QR Attendance
- Notification System
- Calendar Integration
- WhatsApp / Email Automation
- Poster & Reel Generation
- Multi-Organization Support
- Multi-Agent AI

---

## Success Criteria

Version 1 is successful if a club can:

1. Create an event.
2. Manage its budget.
3. Track resources.
4. Record attendance.
5. Upload documentation.
6. Generate a complete event report.
7. Archive the event successfully.

---

## Guiding Principle

> Version 1 should be simple, reliable, and difficult to break.
>
> Intelligence and automation will be added incrementally in future versions.
